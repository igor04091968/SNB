# SNB

Репозиторий с заготовками и утилитами для задач Северного Народного Банка.

Сейчас в репозитории опубликован проект [snb-locations-feed](./snb-locations-feed) для генерации фида по офисам и банкоматам на основе живых данных с `sevnb.ru`.

Также добавлен проект [snb-worktime-webui](./snb-worktime-webui) для расчета рабочего времени сотрудников из Windows session snapshots и подтверждающих activity windows через локальный web GUI.
